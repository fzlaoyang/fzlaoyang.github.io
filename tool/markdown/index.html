Markdown 是一种轻量级标记语言，用简洁的纯文本符号实现排版，无需复杂操作即可快速写出结构清晰的内容。 它支持标题、列表、粗体、链接等常用格式，语法易记（如#表标题、**表粗体），兼容几乎所有编辑器，且可轻松转为 HTML、PDF 等格式。 因高效、易读、跨平台，它成为写作、笔记、文档编辑的主流工具，尤其受程序员、创作者青睐。
