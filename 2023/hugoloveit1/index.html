hugo loveit 主题的美化扩展标签

admonition shortcode 支持12种颜色样式,更好的展现美化您的文章.

支持Markdown或者HTML格式.

admonition的语法

  • type 横幅的颜色样式

  • title 横幅的标题内容

  • open 横幅是否自动展开 默认true

完整代码：

1
2
3
4
5
6
7

{{< admonition type=tip title="示例标题" open=false >}}
示例内容
{{< /admonition >}}
或者
{{< admonition tip "示例标题" false >}}
示例内容
{{< /admonition >}}
